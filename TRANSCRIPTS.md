# Transcripts

## Overview

Episode transcripts are generated from WhisperX transcription output and published as HTML pages.

## Available Transcripts

- [Episode 1: AI Agents Speedrun Civilization on Moltbook](https://ethanjstark.github.io/moltbook-podcast/transcript-episode-1.html)

## Generation Process

Transcripts are generated using the moltbook-report repository:

1. Transcribe audio with WhisperX (or use existing transcription)
2. Run generation script: `npm run generate-ep1-transcript`
3. Copy generated HTML to moltbook-podcast repo
4. Update index.html to add transcript link
5. Commit and push

## Format

- HTML pages with consistent styling
- Full transcript text from audio
- Navigation back to episode page
- Mobile-responsive design

## Note on Speaker Diarization

Episode 1 transcript does not include speaker diarization (all text shown as single speaker). Future episodes may include speaker identification with color-coded labels.
